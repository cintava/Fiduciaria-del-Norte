## Descripción
Sitio web multipágina para **Fiduciaria del Norte S.A.**, empresa de soluciones financieras integrales orientada al desarrollo económico del sector público y privado. El sitio presenta la información institucional, productos y servicios, concursos públicos y preguntas frecuentes.
Proyecto final del curso de **Desarrollo Web** en Coderhouse.

## Demo
🔗 **[Ver sitio en vivo](https://cintava.github.io/Fiduciaria-del-Norte/)**

## Tecnologías
- **HTML5** — Estructura semántica y accesible
- **CSS3** — Estilos personalizados
- **SASS (SCSS)** — Preprocesador CSS con arquitectura modular
- **Bootstrap 5.2.3** — Framework CSS para layout responsive y componentes
- **Figma** — Diseño de la interfaz
- - **AOS** — Animaciones al hacer scroll

## Estructura del proyecto
```
fiduciaria-del-norte/
├── paginas/              # Páginas internas (nosotros, productos, contacto, etc.)
├── css/                  # Estilos compilados
├── scss/                 # Archivos fuente SASS
├── multimedia/           # Imágenes y recursos gráficos
├── bootstrap-5.2.3-dist/ # Framework Bootstrap
├── index.html            # Página principal
├── package.json          # Dependencias
└── README.md
```

## Características
- Sitio multipágina con navegación consistente
- Diseño responsive con Bootstrap 5
- Sección de preguntas frecuentes con acordeones interactivos
- Secciones de concursos públicos con tarjetas informativas
- Arquitectura de estilos modular con SASS
- Animaciones y transiciones CSS

## Páginas
| Página | Contenido |
|--------|-----------|
| Inicio | Presentación institucional, productos destacados y FAQ |
| Nosotros | Historia y equipo de la empresa |
| Productos y Servicios | Sector público y privado |
| Concurso Público | Concursos vigentes, en proceso y culminados |
| Contacto | Formulario de contacto |
